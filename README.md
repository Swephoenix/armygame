# Armygame

A small browser game with an infantry soldier, vehicle entry, mounted aim mode, ammo/reload, and zombie NPCs.

Controls:

- `A` / `D` move the soldier
- `Mouse 1` aim and fire
- `R` reload
- `E` enter or exit the vehicle

This public export includes the runtime HTML, game logic, tests, and the transparent animation frames needed to run the game in the browser.
It excludes raw source videos, large working samples, and other intermediate production assets.
